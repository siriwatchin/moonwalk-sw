# Moonwalk Rehab Cane — Metrics & Biofeedback Reference List

Verified citations behind two questions for the Moonwalk sensorized cane (force/pressure + IMU):
**(1) which gait/weight-bearing metrics drive rehab improvement**, and
**(2) how the biofeedback should be designed**.

All entries confirmed in Semantic Scholar via DOI lookup or title/abstract match. No DOIs fabricated.
Resolve any DOI with `https://doi.org/<DOI>`. BibTeX: `s2cli bibtex <paperId>`.

> Corrections vs. the earlier web brief: WSFC first author is **Kang** (not Oh); Sigrist is S2-dated **2012** (print 2013); the bandwidth-feedback paper PMC3796837 was unverifiable and replaced with **Sadowski 2013**.
